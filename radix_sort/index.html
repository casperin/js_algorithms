Radix sort

Two main functions:

radix_sort
Takes a list and sorts it using the radix algorithm implemented in Javascript (well, Coffeescript)
Example: radix_sort([4, 5, 3, 2, 0, 1, 3]) => [0, 1, 2, 3, 3, 4, 5]
radix_random
Takes two integers as arguments. The first argument is the amount of numbers you want to be sorted, and the second argument is maximum number of digits in each number (more digits = slower sorting).
Example: radix_random(5, 2) => "Random: 31 25 4 86 39 \n Sorted: 4 25 31 39 86"
radix_random uses radix_sort internally.

Notice: This implementation only deals with integers that are equal to, or above, zero.
